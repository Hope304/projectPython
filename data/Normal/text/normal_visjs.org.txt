vis.js vis.js community edition * A dynamic, browser based visualization library. The library is designed to be easy to use, to handle large amounts of dynamic data, and to enable manipulation of and interaction with the data. The library consists of the components DataSet, Timeline, Network, Graph2d and Graph3d. Here is a backup of the old (vis@4.21) website: almende.github.io/vis/ Network Display dynamic, automatically organised, customizable network views. Timeline Create a fully customizable, interactive timeline with items and ranges. Graph3d Create interactive, animated 3d graphs. Surfaces, lines, dots and block styling out of the box. Graph2d Draw graphs and bar charts on an interactive timeline and personalize it the way you want. DataSet Manage unstructured data using DataSet. Add, update, and remove data, and listen for changes in the data. Sponsors This site is maintained by the vis.js community. Help to improve this page!
