FAQ License GDAL GDAL documentation Edit on GitHub Next GDAL GDAL is a translator library for raster and vector geospatial data formats that is released under an MIT style Open Source License by the Open Source Geospatial Foundation. As a library, it presents a single raster abstract data model and single vector abstract data model to the calling application for all supported formats. It also comes with a variety of useful command line utilities for data translation and processing. The NEWS page describes the February 2024 GDAL/OGR 3.8.4 release. See Software using GDAL This documentation is also available as a PDF file, and a .ZIP of individual HTML pages for offline browsing. (The .ZIP also includes that .PDF.)
