WordPress publishing service – simplified publishing WordPress publishing service Web publishing doesn’t need to be complicated. At the University of Melbourne, we make WordPress available for sites that don’t need significant custom features, but do need to be easy to update, and quick to publish. If you need help, start with our guides. If you need assistance, including new site requests, submit a helpdesk ticket. If you already have a site, just open the dashboard to start work.
