Имеется три архива: a) сценарий UUYW_Vorkuta б) летняя фотоподложка Z_UUYW_arround в) зимняя фотоподложка ZZ_UUYW_arround_winter PS: Зима в X-Plane устанавливается отдельным пакетом, за неё в примерах отвечают строки содержащие: --------------- Custom Scenery/1000 autogen_Winter/ Custom Scenery/1000 autogen_World_Winter/ Custom Scenery/1000 decals_Winter/ Custom Scenery/1000 forests_winter/ Custom Scenery/1000 roads_Winter/ Custom Scenery/1000 urban terrain_winter/ Custom Scenery/1000_Winter/ --------------- Тогда scenery_packs.ini для лета будет выглядеть так. Отключаем зимние текстуры и зимнюю фотоподложку, включаем летнюю фотоподложку. Строку с самим сценарием не трогаем. =========================================================== SCENERY_PACK_DISABLED Custom Scenery/1000 autogen_Winter/ SCENERY_PACK_DISABLED Custom Scenery/1000 autogen_World_Winter/ SCENERY_PACK_DISABLED Custom Scenery/1000 decals_Winter/ SCENERY_PACK_DISABLED Custom Scenery/1000 forests_winter/ SCENERY_PACK_DISABLED Custom Scenery/1000 roads_Winter/ SCENERY_PACK_DISABLED Custom Scenery/1000 urban terrain_winter/ SCENERY_PACK_DISABLED Custom Scenery/1000_Winter/ ...Сценарии SCENERY_PACK Custom Scenery/UUYW_Vorkuta/ ...Сценарии ...ОСМ ...Библиотеки ...Фотоподложки SCENERY_PACK Custom Scenery/Z_UUYW_arround/ SCENERY_PACK_DISABLED Custom Scenery/ZZ_UUYW_arround_winter/ ... hd и uhd меши SCENERY_PACK Custom Scenery/zzz_Treelines_Farms_Europe_v2/ SCENERY_PACK Custom Scenery/zzz_uhd_global_scenery1/ SCENERY_PACK Custom Scenery/zzz_hd_global_scenery3/ ============================================================ А для зимы scenery_packs.ini будет таким. Включаем зимние текстуры и зимнюю фотоподложку, отключаем летнюю фотоподложку. Строку с самим сценарием не трогаем. =========================================================== SCENERY_PACK Custom Scenery/1000 autogen_Winter/ SCENERY_PACK Custom Scenery/1000 autogen_World_Winter/ SCENERY_PACK Custom Scenery/1000 decals_Winter/ SCENERY_PACK Custom Scenery/1000 forests_winter/ SCENERY_PACK Custom Scenery/1000 roads_Winter/ SCENERY_PACK Custom Scenery/1000 urban terrain_winter/ SCENERY_PACK Custom Scenery/1000_Winter/ ...Сценарии SCENERY_PACK Custom Scenery/UUYW_Vorkuta/ ...Сценарии ...ОСМ ...Библиотеки ...Фотоподложки SCENERY_PACK_DISABLED Custom Scenery/Z_UUYW_arround/ SCENERY_PACK Custom Scenery/ZZ_UUYW_arround_winter/ ... hd и uhd меши SCENERY_PACK Custom Scenery/zzz_Treelines_Farms_Europe_v2/ SCENERY_PACK Custom Scenery/zzz_uhd_global_scenery1/ SCENERY_PACK Custom Scenery/zzz_hd_global_scenery3/ Используемые сторонние библиотеки Используются следующие библиотеки: - 3D people Library - BeachMaxx Library - BS2001 Object Library - Cemetry library - CDB-Library - Europe Library - Europe Library HD - European vehicles library uwespeed - FF Library - FF Library Extended LOD - FJS Scenery Library - Flags of the World - Flyby_Planes - Ground Texture Library (gt library) - ISDGLibrary - JB Library - MisterX Library - NAPS Library - OpenSceneryX - People_Lib - PM Library - R2 Library - RD Library - RE Library - RescueX_Lib - RuScenery - Sea_Life - PP Library - ShoreLine Library - THE FAIB Aircraft Library - The Fruit Stand Library - The Handy Objects Library - Vehicle Library Extension - Waves Library - World Models - XAirportScenery - XS_Library - ZZ DF-Hard Surface Скачать их все с одной страницы можно по адресу: https://forums.x-plane.org/index.php?/forums/topic/90776-master-list-of-libraries/ Если появляются какие либо новые исключительные библиотеки, то мы стараемся включать их в сценарии Авторы проекта и обратная связь | Все порты на одной странице| Планирование полетов 2016 - 2024
